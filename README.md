I had used tools like Python, Pandas, NumPy, Scikit-learn, and Jupyter Notebook to build a Machine Learning model for CODTECH Internship Task 2 – Predictive Analysis using Machine Learning.

In this project, I developed a classification model using the Netflix Titles dataset to predict the type of content (Movie or TV Show) based on different features. The complete Machine Learning workflow was implemented in a Jupyter Notebook, including data preprocessing, feature selection, model training, prediction, and evaluation.

The project started with loading and analyzing the dataset to understand its structure and identify missing values. Data cleaning techniques were applied to handle incomplete data and prepare the dataset for model training. Relevant features such as release year, rating, duration, country, and category information were selected to improve the prediction process.

Since Machine Learning algorithms cannot directly process text data, categorical features were converted into numerical values using Label Encoding. The dataset was then divided into training and testing sets to evaluate the performance of the model on unseen data.

For prediction, a Random Forest Classification algorithm was implemented due to its ability to handle complex relationships and improve classification performance. The model was trained using the selected features and tested using evaluation metrics such as accuracy score, classification report, and confusion matrix.

This Jupyter Notebook demonstrates the complete predictive analysis pipeline, from data preparation to model evaluation. Through this project, I gained practical experience in data preprocessing, feature engineering, supervised learning algorithms, and evaluating Machine Learning models on real-world datasets.
